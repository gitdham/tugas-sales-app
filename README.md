## Sales App
### description
Basic sales web app with features:
- Manage product categories
- Manage products
- Manage sales

### Installation & Setup

1. Copy .env.exampe as .env

2. run `php artisan key:generate`

3. run `composer install`

4. run `npm install`

5. run `php artisan migrate`

6. run `npm run dev`

7. run `php artisan serve`
